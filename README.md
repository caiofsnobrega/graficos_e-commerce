# graficos_e-commerce
### Tarefa Gráficos - módulo 10
Tarefa realizada em início de curso, nos módulos de análise/visualização de dados.
Proposta: Análise dos dados de uma base de e-commerce do ramo de vestuário, levantamento de dados mais relevantes e criação obrigatória dos gráficos:
- Gráfico de dispersão
- Mapa de calor
- Gráfico de barra
- Gráfico de pizza
- Gráfico de densidade
